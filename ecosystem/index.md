# Ecosystem Overview

Web-3D Ecosystem gồm 3 thành phần chính phối hợp với nhau:

| Thành phần | Thư mục | Trạng thái |
|---|---|---|
| Three.js Engine | `Web-3D/THREEJS/` | ✅ Phase D hoàn thành — 26 modules |
| Babylon.js Engine | `Web-3D/BABYLONJS/` | ⏳ Chưa bắt đầu |
| Factory Pipeline | `Factory/` | ⏳ Blender pipeline đang build |

## Tài liệu

- [Factory Compatibility](/ecosystem/factory-compatibility) — Tool nào tích hợp được với engine nào, định dạng nào
- [Sync Log](/ecosystem/sync) — Lịch sử quyết định và trạng thái workspace
