# web3d-projects — Project Bible Repository

> Kho kế hoạch cho mọi dự án 3D trong Web-3D Ecosystem.
> Mỗi dự án = 1 folder, điền từ `_template/`.
> Docs site: https://docs-3d-ng.vercel.app/projects/

---

## Studio

Repo này vận hành như một **xưởng sản xuất phim** — planning only, không có code.

| Thư mục | Vai trò |
|---|---|
| `_studio/` | Brief cho đạo diễn — CLAUDE.md, phân vai, workflow handoff |
| `_template/` | 8-file project bible template — copy cho mỗi project mới |
| `[project-name]/` | Mỗi project là một production |

→ Claude đọc `_studio/CLAUDE.md` khi mở workspace này.

## Cách dùng

1. Copy `_template/` → `[project-name]/`
2. Điền từng file theo thứ tự 01 → 08
3. Càng chi tiết → workflow thực thi càng nhanh và chính xác
4. Claude đọc xong → viết `HANDOFF.md` → chuyển THREEJS session để build
